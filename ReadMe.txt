Игра Monogolf:

 - Данная игра представляет из себя типичный симулятор гольфа.
 - Игрок натягивает рогатку с шариком, затем отпускает, 
   мяч летит с определенной скоростью в направлении запуска, отскакивая от препятствий, пока не попадет в лунку.
 - Всего есть 5 уровней и у игрока есть 10 жизней на прохождение всех этих уровней.
 - При прохождении уровня игроку начисляются очки и монетки.
 - За монетки можно покупать новые раскраски поля.
 - По очкам в конце игры будет составлен общий рейтинг.
 - При проигрыше (трате всех жизней) игроку будет предложено сыграть заново.
